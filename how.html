IoT Intrusion Detection & People Counter System

A smart, secure way to count people entering/exiting a room and detect unauthorized access – explained simply!

Project Goal (Simple Explanation)

This project creates an automatic door monitor that:

  • Counts how many people are inside a room (like a shop, classroom, or office)
  • Knows if someone enters or exits
  • Acts like a security alarm: when "ARMED", any movement triggers a loud buzzer and red alert
  • When "DISARMED", it just quietly counts people (no alarm)
  • Keeps all data safe with encryption (like a secret code)

Perfect for knowing room occupancy and protecting restricted areas!

1. Overall System – How Everything Connects

Sensors → ESP32 Microcontroller → WiFi → Server → Live Dashboard on your browser

2. How People Counting Works

Two invisible light beams (IR sensors) are placed close together at the door.

  • If someone breaks Beam 1 → ENTRY (People +1)
  • If someone breaks Beam 2 EXIT (People -1)
  • The system only counts if the order is correct – this makes it accurate!

3. How Intrusion (Alarm) Works

You control the alarm with buttons on the dashboard:

  • ARMED Mode: Any movement across the door (or staying too long in one beam) = INTRUSION!
    → Loud buzzer sounds for 3-4 seconds
    → Big red flashing alert on screen
    → Intrusion counter increases
  • DISARMED Mode: No alarm – just counts people normally (safe for public hours)

4. Security – Keeping Data Safe

Every message from the sensor is protected like bank data:

  • Encrypted with AES-128 (turns data into secret code)
  • Checked for tampering (integrity verification)
  • Protected from replay attacks

5. Live Dashboard

You see everything in real-time on any phone/computer:

  • Current people inside
  • Total people entered today
  • Armed/Disarmed status
  • Live log of encrypted + decrypted events
  • Big red alert when intrusion happens

This system is reliable, secure, and easy to understand – great for shops, labs, classrooms, or secure rooms!
